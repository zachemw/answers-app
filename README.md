# Answers

A web app based on *The Book of Answers* — tap or hold the eye to reveal a random answer - 346 answers drawn from the original book.

**Live site:** [answers.zacharyewillis.com](https://answers.zacharyewillis.com)

## Stack

- Vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies
- Hosted on [Vercel](https://vercel.com)
- Auto-deploys on every push to `main`
